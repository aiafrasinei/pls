# VARIABLES

A variable is an abstraction of a memory cell.
The memory cell can be of any size.

Attributes of a variable:

- Name - identifier for a variable
- Type - the type of variable determine the size
- Address - the machine memory address where it is located
- Value - the contents of the memory address
- Scope - where the variable is visible
- Lifetime - time it exists in memory
