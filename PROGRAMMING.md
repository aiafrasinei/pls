# PROGRAMMING

The elements of programming:

* **primitive expressions**

The simplest elements the language is concerned with

* **means of combination**

Compound elements are build from simple ones

* **means of abstraction**

Compound elements can be named and manipulated as units

<hr>

In programming we deal with procedures and data.

Any full programming language should be able to describe primitive data and primitive procedures and should have methods for 

combining and abstracting procedures and data.
